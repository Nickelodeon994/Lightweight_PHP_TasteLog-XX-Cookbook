# Lightweight_PHP_TasteLog-XX-Cookbook
 TasteLog  | XX的食谱 是一个超轻量级的食谱分享社区程序。无需 MySQL 等数据库，基于 PHP + JSON 文件实现用户系统、帖子发布、社交互动、打卡、搜索、管理后台等完整功能。前端响应式设计，注重移动端友好体验。

## ✨ 功能特性

- **用户系统**：注册/登录，自动登录令牌，个人资料编辑
- **帖子管理**：发布带图片或视频的食谱，支持最多 9 个媒体附件，标题+正文，编辑/删除
- **社交互动**：点赞/点踩、收藏帖子、关注用户
- **浏览历史**：记录最近 20 条浏览记录
- **打卡系统**：每日打卡，计算连续打卡天数，日历展示
- **搜索功能**：按标题/内容搜索帖子，结果按相关性排序
- **举报机制**：用户可举报违规帖子，管理员后台处理
- **管理后台**：
  - 用户管理（修改角色/重置密码/删除用户）
  - 帖子管理（查看列表/删除）
  - 举报处理（查看/标记已处理/驳回）
  - 站点设置（标题/图标/允许注册/允许游客浏览/主题模式）
- **主题适配**：支持自动跟随系统、强制亮色/暗色模式
- **响应式布局**：桌面端与移动端适配，移动端深度优化

## 🧰 技术栈

- 后端：PHP 7.4+ (原生，无框架)
- 数据存储：文件系统 (JSON)
- 注意：务必将data/users目录的权限设置为仅本地，否则可能造成信息泄露
